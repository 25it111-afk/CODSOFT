# Titanic Survival Prediction

This project is part of the CodSoft Data Science Internship.

## Objective
To predict whether a passenger survived the Titanic disaster using
machine learning techniques.

## Dataset
Titanic dataset from Kaggle.

## Algorithm Used
- Logistic Regression

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Conclusion
The model successfully predicts passenger survival based on given features.
